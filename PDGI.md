# How rtpv-agropv-explorer follows PDGI

A distributed-solar (rooftop + agrivoltaics) grid-hosting explorer for India. Built on **People's Digital Goods & Infrastructure (PDGI)** principles — people before digital, rights-centric, commons-oriented, transparent ([pdgi.org](https://pdgi.org/)).

This is a public scorecard of how those principles are implemented here, with honest notes on where it falls short. It was generated from a repository audit (2026-07); update it as the code changes.

**License:** MIT · **Live:** https://ashwask.github.io/rtpv-agropv-explorer/

Status key: ✅ implemented · 🟡 partial · ⛳ gap · — not applicable.

## Scorecard

### People before digital — ✅
Frames firm energy and grid equity over vanity nameplate GW.

### Transparency and accountability — ✅
Explicit data caveats and cited baselines; honest 'illustrative' flags.

### Decentralisation and no lock-in — ✅
Single-file client-side app; MIT; no backend or lock-in.

### Free software and the digital commons — ✅
MIT-licensed; fully free and open.

### Privacy — ✅
No analytics or trackers.

### Platform cooperativism — 🟡
Contribution path added (CONTRIBUTING + issue template); no in-app submit.

### Humans in the loop (AI does not cut people out) — —
No AI / LLM component.

### A non-digital alternative — ⛳
Web-only; no print / offline / text fallback.

### Grassroots and reaching the divide-affected — ⛳
English-only ('Tamil Nadu' etc. are data labels, not i18n). Direction: vernacular UI.

### Algorithmic fairness — —
An illustrative visualisation; no fairness-sensitive algorithm.

**Overall:** 5✅ 1🟡 2⛳
