# RTPV + AgroPV — firm-energy & grid-hosting explorer (India)

A single-file, client-side prototype (MapLibre, no backend) exploring India's distributed-solar
landscape through the lens that matters: **not nameplate GW, but firm energy and where the grid can
actually absorb rooftop.**

**Live:** see the GitHub Pages link in the repo's About section.

## Tabs
- **Map** — India by **hosting-data visibility** (the core question: where can the grid absorb rooftop, and where is that data even published?), plus rooftop installed, CUF, firm energy, storage-to-firm and firming cost.
- **Strategy** — the catalytic logic: fund the unowned grid-edge gaps, not the crowded deployment/finance layers.
- **Shifts & markers** — baseline → target trackers toward firm, dispatchable distributed solar.
- **Ecosystem** — an actor canvas mapping value-chain jobs to coverage and the catalytic instrument that fits each gap.

## The core idea
GW is not energy. A 100 MW solar plant at ~22% CUF delivers ~530 MWh/day, not 2,400 — and storage adds
~₹4–6/kWh to firm it. And the binding constraint has shifted from supply to **grid absorption**: hosting
capacity is checked by every DISCOM yet published by ~one state. The data exists; the visibility doesn't.

## Data caveats
This is a prototype. **Rooftop-installed GW are real** (~Sept 2025, public sources); **CUF, firm energy,
storage and firming-cost figures are illustrative** (CUF derived from irradiance; firm = nameplate × CUF).
Hosting-visibility classification reflects published policy/disclosure practice. Ecosystem actor roles are
indicative, not endorsements. Verify against MNRE / CEA / state SLDC data before any decision use.
