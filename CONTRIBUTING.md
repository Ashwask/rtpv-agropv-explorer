# Contributing to rtpv-agropv-explorer

A distributed-solar (rooftop + agrivoltaics) grid-hosting explorer for India — a public good. Corrections, additions and reuse are welcome, in the open.

## Suggest a correction or addition
Spot something wrong, stale or missing? [Open an issue](../../issues/new/choose) with:

- **What** is wrong or missing (the specific value, place or record)
- **The correct value**, if you know it
- **A source** we can verify against (a link, document or dataset)

Disputes about data or rights belong in public, so the record stays version-controlled.

## Contribute code or data
1. Fork the repo and create a branch.
2. Keep changes faithful to sourced data — every value should trace to a citable source.
3. Open a pull request describing the change and its source.

## Provenance & licence
Data is presented as sourced; each record carries its origin. Reuse is under the repository
`LICENSE`. Please don't add proprietary or "all rights reserved" content.
